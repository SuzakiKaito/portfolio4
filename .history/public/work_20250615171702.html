岐阜バンジーの特設サイト カンプ

使用スキル

figma Illustrator Photoshop NEXT.JS CSS

制作時間

40時間

制作人数

1人

企画概要

岐阜のバンジージャンプ体験をより魅力的に。
特設サイトのリデザインを通じて、訪れる人々にバンジージャンプへの興味と挑戦心を喚起する。

工夫した点

・トップに大迫力のバンジー画像を配置し、視覚的に印象を残すデザインにした。
・飛んだ人が自身のジャンプ映像をサイト上で見返せる仕組みを導入。
・実際に現地でバンジー体験を行い、リアルな視点を反映したサイト制作を行った。

受験生の勉強時間サポートアプリ

使用スキル

figma Illustrator Photoshop NEXT.JS CSS

制作時間

25時間

制作人数

2人

企画概要

受験生の勉強効率を高め、志望校合格をサポートするアプリ。
ポモドーロタイマーを活用し、集中と休憩のリズムを整えることで、継続的な学習を支援します。

工夫した点

・シンプルで直感的なUIにより、迷わずすぐに操作できる設計。
・最小限のステップでポモドーロタイマー（25分勉強+5分休憩）を開始可能。
・1セットごとにアプリから応援や感想コメントが届き、モチベーション維持をサポート。

女性商品専用のオンラインショップアプリ

使用スキル

figma Illustrator Photoshop HTML SCSS JavaScript

制作時間

10時間

制作人数

4人

企画概要

女性向けのコスメ・衣服・アクセサリーをランダムに表示し、スクロールで気軽に閲覧できるショッピングアプリ。
気に入った商品はその場で購入可能で、直感的に楽しめる買い物体験を提供します。

工夫した点

・商品をスクロールで流し見できる仕様にし、気軽に楽しめる操作性を実現。
・縦スクロールで商品を切り替え、横スクロールで購入画面へ進む設計により、スクロール操作を最大限に活用。
